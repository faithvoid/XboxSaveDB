# XboxSaveDB
Original Xbox savegame database for sakuraPatcher's "Download Save" feature, primarily aimed at preserving 100% saves for online games such as Project Gotham Racing 2, where users are locked out of certain multiplayer items.

## Supported Games
- Project Gotham Racing 2 - All cars unlocked (JayFNG)
- Mechassault - 100% completion, all mechs unlocked (MATRIXS1ayer)
- Unreal Championship 2 - 100% completion, everything unlocked, gold rank (xownagex)
